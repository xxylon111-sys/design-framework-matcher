# 百家号设计框架文档 (BJH Framework)

## 概述

百家号平台共计 **8种标准框架类型**，覆盖16个页面场景。所有页面共享统一的全局基础结构。

---

## 全局基础模块

| 模块代号 | 模块名称 | 说明 |
|---------|---------|------|
| G-Header | 顶部导航栏 | 品牌Logo + 切换按钮 + 热点横幅 + 通知图标 + 用户头像 |
| G-Sidebar | 左侧导航栏 | 发布按钮 + 分组导航菜单 |

---

## 框架定义

### F1 - 综合仪表盘框架 (Dashboard Framework)

**布局类型：** 三栏布局（左导航 + 主内容多区域网格 + 右侧边栏）

**适配场景：** 平台首页、工作台、运营总览等需要聚合多维信息的综合页面

**结构示意：**
```
┌────────────────────────────────────────────────────┐
│                  顶部导航栏                          │
├──────┬──────────────────────────────┬──────────────┤
│      │  ┌─────────────────────┐    │              │
│      │  │   数据概览卡片       │    │  活动横幅    │
│      │  └─────────────────────┘    │              │
│ 左   │  ┌──────────┬──────────┐   │  收入快览    │
│ 侧   │  │ 评论区   │ 热门任务  │   │              │
│ 导   │  ├──────────┼──────────┤   │  快捷导航    │
│ 航   │  │ 热门话题  │ 重要焦点 │   │              │
│      │  └──────────┴──────────┘   │  学习推荐    │
├──────┴──────────────────────────────┴──────────────┤
```

**组成模块：**

| 模块代号 | 模块名称 | 说明 |
|---------|---------|------|
| F1-M1 | 操作入口区 (Action Entry) | 快捷发布按钮组（发布图文、发布视频、发布动态） |
| F1-M2 | 数据概览卡片 (Data Summary Cards) | 关键指标数字展示（昨日/近期数据） |
| F1-M3 | 多区域内容网格 (Multi-zone Grid) | 2×2网格：评论区 + 热门任务 + 热门话题 + 重要焦点 |
| F1-M4 | 右侧活动栏 (Right Activity Bar) | 连续打卡/活动横幅 |
| F1-M5 | 右侧收入卡片 (Right Income Card) | 淘金收入快览 |
| F1-M6 | 右侧快捷导航 (Right Quick Links) | 常用功能入口（4-6个） |
| F1-M7 | 右侧学习推荐 (Right Learning) | 学习/创作推荐入口 |

**布局类型：** 主内容区分割为左右两栏（列表+详情）

**适配场景：** 消息互动、评论管理、私信管理等需要在列表中选取并查看详情的页面

**结构示意：**
```
┌────────────────────────────────────────────────────┐
│                  顶部导航栏                          │
├──────┬─────────────────────────────────────────────┤
│      │  ┌─────────────────────────────────────┐   │
│      │  │ 全部│图文│视频│小视频│动态│图集│...  │   │
│      │  └─────────────────────────────────────┘   │
│ 左   │  ┌────────────────┬────────────────────┐   │
│ 侧   │  │                │                    │   │
│ 导   │  │  消息列表面板   │    详情面板         │   │
│ 航   │  │  （左 55%）    │   （右 45%）        │   │
│      │  │  · 消息项1     │    完整内容          │   │
│      │  │  · 消息项2     │    回复输入区        │   │
│      │  │  · 消息项3     │    操作按钮          │   │
│      │  │                │                    │   │
```

**组成模块：**

| 模块代号 | 模块名称 | 说明 |
|---------|---------|------|

| F4-M2 | 筛选下拉 (Filter Dropdowns) | 用户类型/评论类型等下拉筛选 |

## 设计规范

### 尺寸规范
- 页面宽度：1440px
- 左侧导航宽度：200px
- 主内容区最大宽度：约920px（有右侧栏时）/ 约1200px（无右侧栏时）
- 右侧边栏宽度：约300px
- 卡片圆角：8-12px

### 色彩规范
- 主色：#4E6EF2（蓝色）
- 背景色：#F5F7FA（浅灰）
- 卡片背景：#FFFFFF
- 文字主色：#1A1A1A
- 文字次色：#666666
- 边框色：#E8ECF3

### 间距规范
